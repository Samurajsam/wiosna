English 

Simple application for adding, editing and deleting location data.
The application was created using the Laravel framework.

How to use:
1) In the project root directory there is a file named "zadanie.sql", it is a database that should be run locally (eg by phpmyadmin)
2) After uploading the database, run the command line from the root folder level and enter the command: php artisan serve
Thanks to it, the server will be launched.
3) The next step is to open the generated address in the browser and add to it /wspolrzednes so that we will be redirected to the main page of the project
Example:
http://127.0.0.1:8000/wspolrzednes


In the STRUCTURE.txt file we will find a detailed description of the project

Polski

Prosta aplikacja do dodawania, edycji i usuwania danych o lokalizacji.
Aplikacja została stworzona przy użyciu frameworka Laravel.

Jak użyć:
1) W katalogu głównym projektu znajduje się plik o nazwie "zadanie.sql", jest to baza danych, która powinna być uruchamiana lokalnie (np przez phpmyadmin)
2) Po wgraniu bazy danych uruchom wiersz poleceń z poziomu folderu głównego i wprowadź polecenie: php artisan serve
Dzięki temu uruchomi się serwer.
3) Kolejnym krokiem jest otwrcie w przeglądarce wygenerowanego adresu oraz dopisanie do niego /wspolrzednes dzięki czemu zostaniemy przekierowani na stronę główną projektu
Przykład:
http://127.0.0.1:8000/wspolrzednes
